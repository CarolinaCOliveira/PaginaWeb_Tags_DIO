4. Falando de TAGS

Voltar ao Topo

Muito importante para a criação de elementos dentro de uma página HTML, temos as tags, segue exemplos das mais utilizadas:

  • i: Deixa o texto em Itálico;
  • strong: Deixa o texto em Negrito;
  • p: Inicia um novo paragráfo;
  • h1: Normalmente utilizado como título de textos;
  • title: Título da página WEB;
  • input: Cria um campo para digitação (Não necessita de tag de fechamento);
  • img: Insere imagem na página HTML (Não necessita de tag de fechamento);

4.1. Atributos Básicos TAGS

É uma propriedade que a TAG pode ter.

4.1.1. Atributos Genéricos:

  • id: Serve para identificar um elemento. Usado no JavaScript;
  • style: Serve para colocar comando CSS direto no elemento;
  • class: Serve para atribuir uma classe e padronizar determinado elemento. Também no CSS;

4.1.2. Atributos Específicos:

  • input type="text": Estamos informando que é para se criar um campo que aceita o tipo texto;
  • input type="number": Estamos informando que é para se criar um campo que aceita o tipo número;
  • input type="color": Estamos informando que é para se criar um campo que aceita o tipo cor;
  • img src="endereçodaimagem.png": Insere uma imagem, podendo ser pasta ou página WEB;
  • img width="250" src="img.png": Width é a largura da imagem inserida;

Voltar ao Topo

TAG que utilizamos para inserir textos.

6. Listas

Voltar ao Topo

São listas dentro do HTML, podendo ser ordenada e não ordenada.

6.1. Listas Ordenadas

Apresenta númeração em sua ordenação.

  • ol: Necessita de tag filhas para compor a criação da lista;
  • li: Item da Lista;

Exemplo:

  1. Item 01
  2. Item 02
  3. Item 03

6.2. Listas Não Ordenadas

Apresenta marcadores em sua ordenação.

  • ul: Necessita de tag filhas para compor a criação da lista;
  • li: Item da Lista;

Exemplo:

  • Item 01
  • Item 02
  • Item 03

7. Links

Voltar ao Topo

Serve para inserir endereço de algum outro lugar no nosso código HTML. É representado pelo "a" Alguns exemplos:

  • TAG (target="_blank"): Usamos dentro do "a" para abrir o link em uma outra aba;
  • TAG (target="title"): Usamos dentro do "a" para aparecer o texto que você quer que apareça ao passar o mouse no link;
